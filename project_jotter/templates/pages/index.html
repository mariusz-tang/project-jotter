What's this?

Project Jotter is a tool for quickly writing down plans. It is built for planning computer programming projects such as websites and video games, but is generic enough to be used as a general-purpose notepad.

The tool is completely free to use for any purpose, and you can export your plans to JSON format if this is helpful to you.

You can share your plans with anyone, but note that this is not intended as a full-blown project management application. It is for the quick drafts or notes you might make before committing to a project.
